###读书汇是什么?
一个在线交流读书心得的非官方组织

可以自由的通过 markdown 编辑读书笔记文件 通过 github 发布到这里

###读书汇的分类

**`经典再现`**  是不是每一次再读经典的某个部分都会有新的感悟 新场景或是新的体会
 
**`新书快读`**  只为了更快更方便的了解最近热点新书

**`读书实验室`**  书不只是拿来读的,更是拿来用的,show出实践案例

**其他 `没有预见`到的...**
